# QBZ Downloader v4.0.3

Release date: March 31, 2026

## Highlights

- Desktop-first experience with a polished onboarding flow before entering the dashboard.
- Runtime configuration migrated to local app storage, reducing reliance on environment files during daily use.
- Improved desktop UX with GitHub shortcuts in onboarding, splash screen, and dashboard top bar.
- Stable installer and portable release pipeline for Windows.

## What is new

- Added a dedicated first-run setup gate for required Qobuz credentials.
- Upgraded setup into a two-stage onboarding flow:
  - Intro/onboarding stage.
  - Credentials + download path stage before entering dashboard.
- Added direct GitHub project entry points in:
  - Startup splash screen.
  - Onboarding UI.
  - Dashboard header (next to theme controls).

## Improvements

- Enforced EXE-first runtime path and disabled web entry mode in desktop builds.
- Cleared stale service worker and cached PWA artifacts in EXE mode to prevent old UI from reappearing.
- Updated README to focus on desktop usage and build outputs:
  - `npm run desktop:dist` (Installer)
  - `npm run desktop:dist:portable` (Portable)
  - `npm run desktop:dist:all` (Both)

## Migration notes

- Existing users should review values in desktop Settings after upgrading.
- New users can complete setup directly from in-app onboarding (no repository clone required for normal usage).
