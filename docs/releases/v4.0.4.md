# QBZ Downloader v4.0.4

Release date: March 31, 2026

## Summary

This patch release improves desktop reliability and clarifies product messaging for EXE-first usage.

## Highlights

- Fixed dynamic download path behavior in desktop runtime.
- Improved `/downloads` route handling so path updates are reflected without restarting.
- Updated project wording to reflect desktop-first positioning.

## Fixed

- **Issue #4**: changing download path in Settings could lead to confusion where completed downloads were not visible in the newly selected folder.
  - New downloads now use the latest configured path immediately.
  - Existing files remain in their previous folder (no automatic move).

## Docs and Messaging

- README updated to desktop-first language and clearer onboarding wording.
- Added explicit note about download path behavior:
  - path changes apply immediately for new downloads
  - previous files are not relocated automatically

## Release strategy

- Older stable releases (`v4.0.3`, `v4.0.0`) are intentionally kept for rollback and troubleshooting history.
